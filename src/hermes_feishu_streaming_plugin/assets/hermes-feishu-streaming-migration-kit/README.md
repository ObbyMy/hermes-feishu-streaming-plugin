# Hermes Feishu Streaming Migration Kit

Feishu-only migration bundle for the Hermes Agent gateway experience.

This bundle preserves:
- single editable Feishu streaming card output
- embedded tool progress inside the active card
- status-only / metadata-only stream updates
- session-reset and context-pressure notice cards
- caption-first then native file send behavior

Use the standalone plugin CLI to inspect, export, locate, and apply this bundle to another Hermes repo.
